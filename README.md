# Near Buy – Find Products Near You

**"Near Buy"** – A simple, fast, and privacy-first web app that helps Filipinos find the **best prices on groceries, fruits, and essentials — from stores *near you*.**

> **"Bakit malayo? Near Buy!"**

---

## Features

| Feature | Description |
|--------|-------------|
| **Search Near Me** | Uses your GPS to show products within your chosen radius |
| **Real-Time Distance** | See exactly how far the store is (in km) |
| **Best Price Highlight** | Instantly spot the cheapest deal in your area |
| **Interactive Map** | View store location, get directions, and see your search radius |
| **Favorites System** | Save products to revisit later |
| **Store Owner Dashboard** | Add, edit, delete products & track inventory |
| **Admin Panel** | Manage users, stores, and system stats |

## Tech Stack

- **Frontend**: HTML, Tailwind CSS, Vanilla JavaScript, Leaflet.js (OpenStreetMap)
- **Backend**: PHP 8+, MySQL
- **Location**: HTML5 Geolocation API + Haversine Formula
- **No frameworks. No trackers. No ads.**
